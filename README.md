# LearnXpert Landing Page

## 📌 Project Description

This project is a landing page built from a design mockup for an online learning platform (AI / digital services).

The main goal of the project is to practice layout development, responsive design, and working with interactive elements.

## 🚀 Demo

👉 https://danabol100.github.io/learnxpert-landing/

## 🛠 Technologies Used

* HTML5
* SCSS (Sass)
* JavaScript (Vanilla JS)

## 📱 Features

* Responsive layout (desktop / tablet / mobile)
* Modern UI (landing page with services, benefits, and CTA sections)
* Interactive elements (menu, buttons, hover effects)
* Semantic and well-structured markup

## 🎯 Project Highlights

* Built from a design mockup
* SCSS used for better style organization
* Clean and maintainable code structure
* Optimized for different screen sizes

## 📂 Project Structure

* /css — styles
* /js — scripts
* /img — images
* index.html — main page

## 🛠 Installation and Setup

1. Clone the repository:
   git clone https://github.com/danabol100/learnxpert-landing.git

2. Open the index.html file in your browser
